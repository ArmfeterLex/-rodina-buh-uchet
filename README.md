# -rodina-buh-uchet
Бухгалтерская система для сельскохозяйственного предприятия СХА «Родина» на C# Windows Forms. Включает модули учета контрагентов, материально ответственных лиц (МОЛ) и доверенностей, формирования отчетности за период, решения транспортной задачи, авторизации с ролевой моделью доступа. База данных SQLite.
